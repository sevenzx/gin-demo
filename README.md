## gin-demo

内含gin、gorm、gorm-gen的快速学习，以及使用它们来构建一个todo清单的demo



### 项目结构

```
├── controller			
├── db
│   ├── gen			gen配置	
│   ├── mysql.go		MySQL配置
│   └── todo.sql			
├── main.go			主入口
├── model							
│   ├── entity
│   └── vo
├── query
│   ├── gen.go
│   └── todo.gen.go
├── route
│   └── route.go		路由配置
├── static
├── study			快速学习
│   ├── gen
│   ├── gin
│   └── gorm
└── templates
```
